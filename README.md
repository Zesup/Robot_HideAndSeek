# Robot_HideAndSeek
Cache-cache dans une pièce contre un robot

La librairie qui sera utilisé sera OpenCV.
Le langage utilisé sera C++.

Matériel nécessaire :
- caméra type raspberry
- raspberry pi
- jetson nano
- servomoteur(s)
- carte interface I2C/PWM
- support mécanique
- alimentation externe pour le(s) servomoteur(s)
- objets à détecter
